# AndroidSkin
一个使用成本极低的换肤框架 

- 支持插件式换肤
- 支持继承和非继承式换肤，无需对base层进行代码侵入
- 支持自定义View
- 支持多种加载类型，Assets，Storage 等
- 内部直接进行换肤，无需重启
